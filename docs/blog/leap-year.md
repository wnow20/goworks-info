# 闰年(Leap Year)产生的原因及判断方法

## 原因
总结一句话
是因为地球公转时间与公历每年总时间的差异，差异逐年累加，每四年就会多一天。

地球公转天数：365.24219
公历平年天数：365

如果按公转天数按365.25算，4年闰一天，即：
365 * 4 + 1(2月29日) == 365.25 * 4

但是365.25 与实际公转时间还是有误差，
365.25 - 365.24219 == 0.00781

差不多是每四百年，早过了三天，得扣回来。

因此，规则是百年不闰，每四百年闰一天。


## 判断规则
年份是4的倍数，但不是100的倍数，为闰年
年份是400的倍数，为闰年
